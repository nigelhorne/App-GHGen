  188: the test run and before Perl::Critic.  The step installs L<warnings::unused>
  189: from CPAN and runs C<perl -Mwarnings::unused -c> on every C<.pm> file under
  190: C<lib/>.

  192: This step is conditioned on the latest matrix Perl version and
  193: C<ubuntu-latest>, matching the Perl::Critic and coverage steps.  It is also
  194: marked C<continue-on-error: true> because unused-variable warnings can be
