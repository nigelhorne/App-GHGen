# NAME

App::GHGen - GitHub Actions workflow generator and analyzer

# SYNOPSIS

    use App::GHGen;
    
    # Command-line usage:
    ghgen generate --type=perl
    ghgen analyze

# DESCRIPTION

App::GHGen helps you create and optimize GitHub Actions workflows. It can:

- Generate optimized CI/CD workflows for various languages
- Analyze existing workflows for performance and cost issues
- Suggest improvements and fixes

# SUPPORTED LANGUAGES

- Node.js/npm
- Python
- Rust
- Go
- Ruby
- Perl
- Docker
- Static sites (GitHub Pages)
